Glosario de terminos de POO

Algoritmo

Ver más >>
Un grupo finito de operaciones organizadas de manera lógica y ordenadaque permite solucionar un determinado problema.

Programación

Ver más >> acción de crear programas a través del desarrollo de un código, que se basa en el conjunto de instrucciones que sigue el ordenador para ejecutar un programa.

Lenguaje de Programación

Ver más >> un lenguaje diseñado para describir el conjunto de acciones consecutivas que un equipo debe ejecutar.

Paradigma

Ver más >> un modelo o patrón sostenido en una disciplina.

Paradigma Structurado o Secuencial

Ver más >> una técnica que utiliza la metodología secuencial de desarrollo de programas

Programación orientada a objetos

Ver más >> una forma de programar que trata de encontrar una solución a problemas a través de objetos.

Clase

Ver más >> plantilla para la creación de objetos de datos según un modelo predefinido.

Objeto

Ver más >> unidad dentro de un programa informático que tiene un estado, y un comportamiento.

Atributo

Ver más >> caracterísiticas individuales que diferencian un objeto de otro y determinan su apariencia, estado u otras cualidades.

Método

Ver más >> procedimientos y funciones que se invocan para actuar sobre los objetos.

Evento

Ver más >> medio como interactúa una clase con otras o con el propio usuario, se encargan de avisar que algo ha ocurrido y de manejarlo de una forma o de otra.

Encapsulamiento

Ver más >> ocultamiento del estado, es decir, de los datos miembro de un objeto de manera que solo se pueda cambiar mediante las operaciones definidas para ese objeto.

Abstracción

Ver más >> método que se usa al momento de analizar un elemento particular, despreciando los aspectos no relevantes para el estudio específico.

Herencia

Ver más >> propiedad que permite que los objetos sean creados a partir de otros ya existentes.

Polimorfismo

Ver más >> capacidad que tienen los objetos de una clase de responder al mismo mensaje o evento en función de los parámetros utilizados durante su invocación.

Modulación

Ver más >> consiste en dividir un programa en módulos o subprogramas con el fin de hacerlo más legible y manejable.

Minecraft

Ver más >> videojuego de mundo abierto donde la exploración y las construcciones son fundamentales.
